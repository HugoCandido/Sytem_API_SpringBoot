# 📌 Gerenciamento de usuarios

Desenvolvimento de uma organização e classificação de usuarios ou clientes, sendo visualizada somente pela parte do nack-end.

## 🚀 Tecnologias utilizadas
- Java
- Spring Boot


## 🎯 Funcionalidades
✅ Cadastro de usuários
✅ Consulta de dados via API externa
❌ Integração com banco de dados (desenvolvimento)

## 🧠 Aprendizados
#### Esse projeto é desenvolvido com foco em aprendizado de:

- Integração com APIs REST
- Manipulação de dados em Java
- Persistência com banco de dados
- Boas práticas de organização de código
